# Code of Conduct

This project expects respectful, constructive, and evidence-oriented collaboration.

## Expected behavior

- Stay within the repository's stated ownership boundaries.
- Prefer precise, reviewable claims over unsupported assertions.
- Treat contributors and reviewers with respect.
- Keep discussion focused on the technical scope of the change.
- Route sensitive operational or security material through the security process rather than public issues.

## Not acceptable

- Harassment or discriminatory conduct.
- Publishing secrets, credentials, or private operational data.
- Misrepresenting readiness, certification, licensing, or security posture.
- Bypassing review or validation gates.

## Maintainer action

Maintainers may close, edit, lock, or remove contributions that violate this code or fall outside the repository scope.
